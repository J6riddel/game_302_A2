## Project title
Do you copy?

## Group Number
13A

## Description
Brief overview of your game’s concept, mechanics, and player
experience

"Do you copy? challenges the player to go through a dark maze and find their way out while decoding directions from Commander Static from static walkie talkie.  The experience emphasizes listening struggle and raises awareness about the hard of hearing disability through a maze and dark aesthetics and audio distortion.

## Setup and Interaction Instructions
How to run and play the game
Players navigate with arrows, if they select the correct direction, the maze slightly brightens. If the player can not understand the prompt, they can use 5% of their battery to clarify it, but the player needs to be strategic about it; because if the walkie talkie dies they will be stuck in the maze forever.

## Detailed instructions

1. Click Start or Tutorial for a detailed explanation of the game meachanics
2. Use Arrows to navigate character
3. Static audio, you have to listen carefully. To clarify voice it costs 5% of your battery
4. Follow the directions to exit, interface brightens if successful, darkens if player loses
5. Battery starts at 100%, once it reaches 0%, player dies
